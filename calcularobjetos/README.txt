USAR EL PROGRAMA

una vez iniciado el programa este te pedira que ingreses la medida de un lado de tu objeto:
|----------------------------|
|Ingresar la medida del lado:|
|----------------------------|

una vez ingresado el numero 
por ejemplo: 3
se desplegara el siguiente menu


|---------------------------------|
|Dato actual: 3.0                 |
|1.- Calcular Area                |
|2.- Calcular perimetro           |
|3.- Calcular volumen             |
|4.- Calcular perimetro(cubo)     |
|5.- Cancelar                     |
|6.- Salir                        |
|---------------------------------|

una vez aqui puede seleccionar cualquiera de las opciones para que se realice la operacion correspondiente

al finalizar alguna de las operaciones, se desplegara un menu como el siguiente:
|-----------------------------|
| Dato actual: 3.0            |
| Elige una opcion:           |
|1.-Ingresar nuevo dato       |
|2.-Seleccionar otra operacion|
|-----------------------------|

Escribe 1 para ingresar un nuevo valor o
Escribe 2 para seguir utilizando el dato actual con otra operacion

|---------------------------------|
|Dato actual: 3.0                 |
|1.- Calcular Area                |
|2.- Calcular perimetro           |
|3.- Calcular volumen             |
|4.- Calcular perimetro(cubo)     |
|5.- Cancelar                     |
|6.- Salir                        |
|---------------------------------|

opcion 5: cancela el proceso y te pide nuevamente un valor para el lado de la figura
opcion 6: Finaliza el programa
