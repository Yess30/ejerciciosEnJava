EJECUTAR
Para ejecutar el proyecto en cmd hacer lo siguiente:
desde cmd entrar a la ubicacion de la carpeta, en mi caso:
C:\Users\yessp\Desktop>cd C:\Users\yessp\Desktop\ejerciciosJava\traductorMorse

1.-
Ahi escribimos el siguiente comando
"javac TraductorMorse.java"

2.-
y despues el comando "java TraductorMorse.java"

esto ejecutara el programa

USAR EL PROGRAMA

una vez iniciado el programa desplegara un menu como el siguiente:

|---------------------------------|
|Que accion desea hacer           |
| 1.- Traducir de morse a texto   |
| 2.- Traducir texto a morse      |
| 3.- Salir                       |
|---------------------------------|

con base a los numeros escribir su seleccion
si selecciona la opcion 1:
Debe ecribir un mensaje en clave morse, teniendo especial cuidado en que cada letra tenga un espacio
y cada palabra tres espacios
Por ejemplo:
".... --- -.--   . ...   .-.. ..- -. . ..."

el programa procedera a traducir la oracion
y mostrara el resultado, en este caso:
"hoy es lunes"

al concluir el programa volvera a lanzar el menu del inicio:

|---------------------------------|
|Que accion desea hacer           |
| 1.- Traducir de morse a texto   |
| 2.- Traducir texto a morse      |
| 3.- Salir                       |
|---------------------------------|

Si ahora selecciona la opcion 2 debera introducir su oracion en texto normal
por ejemplo:
"hoy es lunes"
 y el programa la transformara a clave morse
 este seria el resultado:
".... --- -.--   . ...   .-.. ..- -. . ..."

nuevamente el programa mostrara el menu de inicio


|---------------------------------|
|Que accion desea hacer           |
| 1.- Traducir de morse a texto   |
| 2.- Traducir texto a morse      |
| 3.- Salir                       |
|---------------------------------|

puede nuevamente seleccionar alguna de las opciones del traductor 
o seleccionar la opcion 3.- Salir
esto finalizara el programa
